• Gene Topology:

The International Cell Processing Company (ICPC) is a world leader in the analysis of genetic sequences. A genetic sequence is a string containing only the letters A, C, G, and T, each representing a nucleotide (Adenine, Cytosine, Guanine, and Thymine, respectively).

ICPC has developed a technique called Genetically Optimized Organic Folding (GOOF) to simplify genetic sequences. The technique identifies specific folding points within a sequence where the sequence reads the same in both directions from the point of folding. Folding at these points merges identical nucleotides, resulting in a simplified sequence.

Our project automates the GOOF process to identify optimal folding points and obtain the shortest possible sequence. This automation helps streamline the analysis and reduces manual effort in finding folding points.

• AI-based Voice Assistant (Ongoing):

This project involves creating an AI-based voice assistant using Python and various machine learning libraries. The assistant is designed to understand and respond to voice commands, making use of natural language processing techniques to provide accurate and useful responses.

The project is currently ongoing, and the aim is to enhance the assistant's capabilities, including better voice recognition, more sophisticated responses, and integration with various APIs to provide additional functionalities.
